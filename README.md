# :hamburger: 我的餐廳清單 :hamburger:
一個可以讓你查看我最愛的餐廳清單的網站。

## :pencil2: 功能
- 使用者可以在首頁看到所有餐廳與它們的簡單資料：
  - 餐廳照片
  - 餐廳名稱
  - 餐廳分類
  - 餐廳評分
- 使用者可以再點進去看餐廳的詳細資訊：
  - 類別
  - 地址
  - 電話
  - 描述
  - 圖片
- 使用者可以透過搜尋餐廳名稱來找到特定的餐廳
